# Features

## Blog

- Markdown posts in `src/content/blog/`
- Schema: `title`, `description`, `pubDate`, `category`, `tags[]`, `image?`, `draft`
- Route: `/blog/[slug]/` — slug is the filename without `.md`
- Filtered from build when `draft: true`
- Read time calculated at build time (~200 words/min)
- RSS feed at `/rss.xml` — auto-generated from all non-draft posts
- RSS autodiscovery `<link>` in every page `<head>`
- RSS icon link in blog index header

## RSS Feed

- Generated by `src/pages/rss.xml.js` using `@astrojs/rss`
- Feed URL: `https://www.antoniocumberbatch.com/rss.xml`
- Includes: title, description, pubDate, categories/tags, link per post

## Visit Counter

- Azure Function: `GetResumeCounter` (C#, Cosmos DB binding)
- Cosmos DB: `AzureResume` database, `Counter` container, document `id="1"`
- Returns pre-increment count; function increments on read
- Displayed on homepage: `<span id="visit-counter">`
- Fetched by `main.js` on page load; animates in via `window.animateCounter()`

## Dark Mode

- Implemented via `data-theme` attribute on `<html>`

## Image Optimisation

Local images are in `src/assets/images/` so Astro's build pipeline:
- Converts PNG/JPG → WebP
- Resizes to the declared `width`/`height` on the `<Image>` component
- Outputs content-hashed files to `dist/_astro/`

| Image | Original | Optimised |
|-------|----------|-----------|
| me.png (hero photo) | 4,740 KB | 12.7 KB WebP |
| nicole-cumberbatch-sassi.jpg | 4,307 KB | 2.2 KB WebP |
| sassi-team-photo.jpg | 3,488 KB | 47.6 KB WebP |
| sassi-client-portal.png | 184 KB | 12.8 KB WebP |
| sassi-admin-portal.png | 182 KB | 10.8 KB WebP |

Remote images (Microsoft cert badges from `learn.microsoft.com`) are fetched and converted at build time. Allowed via `image.remotePatterns` in `astro.config.mjs`.
