Overview

Jimney Shop is a growing retail and e-commerce business seeking to establish a reliable, secure, and scalable technology foundation. ARC Cloud Consulting was engaged to design and deliver their end-to-end IT and cloud infrastructure — spanning networking, cloud hosting, security, and operational tooling.

The project covered everything from initial infrastructure assessment through to live deployment, ensuring the business had a production-ready cloud environment capable of supporting their digital commerce ambitions from day one.

The Challenge

  • No existing cloud infrastructure — the business was operating entirely on local and ad-hoc systems.
  • No formal IT security posture — devices, networks, and data lacked consistent protection controls.
  • E-commerce aspirations required PCI-DSS-aware payment processing infrastructure.
  • Need to integrate online storefront with back-office inventory and order management systems.
  • Small team with limited in-house IT capability — solution needed to be manageable post-handover.

The Solution

ARC Cloud Consulting designed a cloud-first infrastructure on Microsoft Azure, purpose-built for e-commerce operations. The architecture prioritised security, reliability, and ease of management — ensuring the Jimney Shop team could operate confidently with minimal IT overhead.

  • Azure Virtual Network design with subnet segmentation for web, application, and data tiers.
  • Azure App Service deployment for the e-commerce storefront with autoscaling configured.
  • Azure SQL Database for product catalogue, order management, and customer data storage.
  • Azure Key Vault for secure storage of API keys, connection strings, and payment gateway credentials.
  • Microsoft Defender for Cloud enabled across all resources for continuous security assessment.
  • Azure Backup configured for all critical data with geo-redundant replication.
  • Staff device management via Microsoft Intune — policies for secure access and endpoint protection.
  • End-user training and full operational documentation delivered at project close.
